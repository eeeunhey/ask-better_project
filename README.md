# Prompt Tailor ✂️  
> "지금 궁금한 걸 편하게 적어보세요"  
> 사용자는 대충 적어도, 뒤에서는 깔끔한 프롬프트로 변환해주는 **프롬프트 생성기**입니다.

---

## ✨ 주요 기능

### 1) 메인 화면 (프롬프트 생성기)
- 상단 질문 입력 박스  
  - 플레이스홀더: **"지금 궁금한 걸 편하게 적어보세요"**
  - 사용자는 말하듯이 막 적어도 됨 (반말/띄어쓰기 엉망 OK)

- 옵션 영역 (필터/태그)
  - **목표**: `개념설명 / 계획짜기 / 코드디버깅 / 글고치기 / 기타`
  - **톤**: `친절 / 압축 / 직설 / 초등학생 수준`
  - **출력 형식**: `리스트 / 단계 / 표 / 코드`

- 생성 버튼
  - 버튼 텍스트:
    - `프롬프트 만들기` 또는  
    - `AI에게 잘 묻는 버전으로 바꾸기`

### 2) 결과 영역 (좌/우 레이아웃)
- **왼쪽**: 사용자가 입력한 원래 질문 그대로 보여주기
- **오른쪽**: Gemini가 생성한 “추천 프롬프트”
- 하단 버튼
  - `Copy` / `복사하기`
  - `Edit` / `살짝 수정하기` → 텍스트 영역으로 변경 후 수정 가능

---

## 🧱 기술 스택

### Frontend
- React (Vite 또는 CRA)
- TypeScript (선택)
- 상태 관리: React 훅 기반 (useState, useEffect 등)
- 스타일:  
  - 선택 1) Tailwind CSS  
  - 선택 2) MUI

### Backend
- Node.js + Express
- Google Gemini API 연동 (REST 호출)
- CORS 설정 / 환경변수 관리
